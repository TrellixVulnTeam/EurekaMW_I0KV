1) Checkout the code.
2) Install pip3. 
3) Install flask.
4) Install pymongo.
5) Install dnspython.
6) Open the terminal in the root directory of the project.
7) Run the following cmd:
	a) For website: python -m com.eurekamw_mg.controller.Main
	b) For Rest servces: python -m com.eurekamw_mg.REST.RestServices
